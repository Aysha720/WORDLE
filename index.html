Game Rules


  1. Each guess must be a valid five-letter word.
  2. The color of a tile will change to show you how close your guess was.
  3. If the tile turns green, the letter is in the word, and it is in the correct position.
  4. If the tile turns yellow, the letter is in the word, but it is not in the correct position.
  5. If the tile turns gray, the letter is not in the word.
